# PlotEllipse
Plot an ellipse in s2map around two GPS locations.

Example usage: 

> python plot_ellipse.py --p1_lat 32.076761 --p1_lng 34.792510 --p2_lat 32.083257 --p2_lng 34.767737 -r 3000
